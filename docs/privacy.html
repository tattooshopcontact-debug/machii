Politique de confidentialité

Dernière mise à jour : 10 juin 2026

Machii (« nous », « notre », « l'application ») est une application mobile de covoiturage gratuit destinée à mettre en relation des conducteurs et des passagers en Tunisie. Cette politique de confidentialité décrit comment nous collectons, utilisons et protégeons les données personnelles que tu nous confies.

1. Données collectées

1.1 Données que tu nous fournis directement

  • Numéro de téléphone : pour créer ton compte et te connecter via un code de validation.
  • Prénom : pour t'identifier auprès des autres utilisateurs.
  • Photo de profil (optionnelle) : si tu choisis d'en ajouter une.
  • Rôle (passager / conducteur / les deux) : pour adapter ton expérience.
  • Bio (optionnelle) : informations libres que tu choisis de partager.
  • Documents de vérification d'identité (CIN, permis de conduire, carte grise, photo de véhicule) : uniquement si tu demandes la validation conducteur. Ces documents sont stockés chiffrés et ne sont jamais partagés avec les autres utilisateurs.
  • Contacts d'urgence : nom et numéro de téléphone des personnes que tu choisis d'ajouter pour le bouton SOS.

1.2 Données techniques

  • Géolocalisation (uniquement pendant l'utilisation) : pour afficher les trajets proches et pour le bouton SOS. Jamais en arrière-plan.
  • Token de notification push : pour t'envoyer les alertes de réservation et de message.
  • Logs techniques : type d'appareil, version OS, erreurs applicatives — pour améliorer la stabilité.

1.3 Données générées par ton activité

  • Trajets que tu publies et réservations que tu effectues.
  • Messages échangés avec d'autres utilisateurs via le chat interne.
  • Notes et commentaires laissés après un trajet.

2. Finalités du traitement

Tes données sont utilisées pour :

  • Te connecter à l'application (numéro de téléphone + code de validation).
  • Mettre en relation passagers et conducteurs (recherche de trajets, réservations, chat).
  • Sécuriser les trajets (bouton SOS, partage de position à tes contacts d'urgence en cas d'alerte).
  • Vérifier l'identité des conducteurs (KYC documentaire, optionnel).
  • T'envoyer des notifications fonctionnelles (demande de réservation reçue, acceptation, nouveau message).
  • Améliorer le service (analyse anonymisée d'usage, correction de bugs).

Aucune donnée n'est utilisée à des fins publicitaires.

3. Base légale

Le traitement de tes données repose sur :

  • L'exécution du contrat que constitue l'utilisation de l'application.
  • Ton consentement pour les permissions géolocalisation, notifications et accès photos.
  • Notre intérêt légitime à sécuriser la plateforme et prévenir les abus.

4. Partage de données

Nous ne vendons jamais tes données. Nous les partageons uniquement avec :

  • Les autres utilisateurs de Machii (prénom, rôle, photo, note moyenne — visibles dans tes interactions ; ton numéro de téléphone est révélé au conducteur uniquement après acceptation explicite de la réservation).
  • Nos sous-traitants techniques :
    • Supabase (hébergeur de la base de données et de l'authentification, données hébergées sur serveurs UE).
    • Expo Application Services / EAS Push (envoi des notifications push).
    • Meta WhatsApp Cloud API (envoi du code de validation par WhatsApp, lorsque ce mode sera activé).

Aucune donnée n'est partagée avec des tiers à des fins commerciales.

5. Durée de conservation

  • Compte actif : tant que tu utilises l'application.
  • Compte inactif : 24 mois sans connexion → suppression automatique.
  • Documents KYC : 12 mois après l'arrêt d'activité conducteur.
  • Messages de chat : conservés tant que la conversation existe ; supprimés avec ton compte.
  • Logs techniques : 90 jours maximum.

6. Tes droits

Conformément au RGPD et à la loi tunisienne n° 2004-63 relative à la protection des données personnelles, tu disposes des droits suivants :

  • Accès à tes données.
  • Rectification de données inexactes.
  • Suppression (« droit à l'oubli ») : ton compte et toutes tes données associées sont supprimés sur simple demande.
  • Portabilité : export de tes données dans un format machine.
  • Opposition au traitement.
  • Retrait du consentement à tout moment.

Pour exercer ces droits, écris-nous à : support@machii.app

Tu peux aussi déposer une réclamation auprès de :

  • INPDP (Tunisie) : https://www.inpdp.tn
  • CNIL (France/UE) : https://www.cnil.fr

7. Sécurité

Toutes tes données sont :

  • Chiffrées en transit (HTTPS / TLS 1.2+).
  • Chiffrées au repos côté base de données (Supabase AES-256).
  • Protégées par Row Level Security (chaque utilisateur ne peut lire que ses propres données).
  • Anti-contournement chat : les numéros de téléphone, emails et liens externes échangés dans le chat avant acceptation sont automatiquement bloqués pour empêcher les arnaques hors plateforme.

8. Mineurs

Machii est interdit aux moins de 18 ans. Si tu nous signales un compte de mineur, nous le supprimerons immédiatement.

9. Modifications

Toute modification substantielle de cette politique te sera notifiée dans l'application au moins 15 jours avant son entrée en vigueur.

10. Contact

  • Éditeur : Machii (Tunisie)
  • Email : support@machii.app
  • Cadre légal : application de covoiturage gratuit conforme à la loi tunisienne n° 2004-33 relative à l'organisation des transports terrestres.
